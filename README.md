## Installation
1. Install Docker Desktop from https://www.docker.com/products/docker-desktop
2. Clone this repo to your Desktop
3. Run: `docker compose -f docker-compose.simple.yml up -d`
4. Load the `extension` folder in Chrome as an unpacked extension
5. The setup wizard opens automatically!
